# PRODIGY_WD_05

## Weather App

This project is developed as part of my Web Development Internship at Prodigy InfoTech.

## Project Description

A weather application that fetches weather data from a weather API based on user input location. It displays current weather conditions, temperature, humidity, and wind speed in an interactive and user-friendly interface.

## Technologies Used

- HTML
- CSS
- JavaScript
- Weather API

## Features

- Search weather by city name
- Displays current temperature
- Shows weather conditions
- Displays humidity and wind speed
- Responsive design
